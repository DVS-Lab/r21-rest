# Covariate Randomise Summary

Generated by `code/check_covariate_randomise_results.py`. FSL corrected-p images store `1-p`; values greater than 0.95 correspond to corrected p-values below 0.05. The summary includes C1/C2 mean-effect tests and C3/C4 covariate-effect tests from the four-row covariate `design.con` files. Significant maps are copied here with JSON sidecars and ROI-value TSVs that join subject-level contrast betas to the covariate audit table. For significant C1/C2 intercept tests, a second stage-2 beta timeseries TSV is written for four-condition bar plots.

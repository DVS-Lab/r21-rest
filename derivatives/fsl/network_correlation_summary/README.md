# Network Correlation Summary

Generated by `code/MakeNetworkCorrelationTables.py` from dual-regression stage-1 timecourse files. The run-values table contains Pearson full correlations and precision-matrix partial correlations among Smith09 network timecourses. The condition-contrast table applies the same within-participant condition contrasts used for the randomise analyses to Fisher-z-transformed correlations. The summary table reports one-sample means and deterministic Monte Carlo sign-flip p-values for each contrast.

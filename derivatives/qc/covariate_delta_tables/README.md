# Covariate Delta Tables

Generated by `code/MakeCovariateDeltaTables.py` from
`derivatives/qc/task-rest_group_covariates.tsv`.

The combined file is `task-rest_covariate_deltas.tsv`; the remaining TSV files
split the same rows into one table per contrast.

By default, the main tables contain only rows with complete blink, pupil, and
mean-FD deltas, so they should not contain `n/a` values. Incomplete rows are
kept separately in `task-rest_covariate_delta_missingness.tsv`. Use
`--include-incomplete` only for debugging.

The default `primary` contrast set contains the five requested covariate checks
plus the two direct BOTH-versus-single-site comparisons:

- `both-minus-sham`
- `both-minus-rtpj`
- `both-minus-vlpfc`
- `both-minus-mean-rtpj-vlpfc`
- `vlpfc-minus-rtpj`
- `vlpfc-minus-sham`
- `rtpj-minus-sham`

`complete_conditions` indicates whether all runs needed for that contrast are
present. `complete_delta_covariates` additionally requires non-missing deltas
for blink rate, pupil area, and mean FD.
